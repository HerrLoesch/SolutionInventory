# Solution Inventory PWA

Dieses Repository enthält ein Starter-Template für eine Progressive Web App mit Vue 3 und Vuetify. Ziel: Fragen in einem Workflow stellen, zwischen Kategorien wechseln und am Ende eine Excel-Datei (.xlsx) exportieren.

Schnellstart:

1. Abhängigkeiten installieren:

```bash
npm install
```

2. Dev-Server starten:

```bash
npm run dev
```

3. Öffne die angezeigte URL im Browser. Die App bietet Beispielkategorien und eine Export-Schaltfläche.

Hinweis: Ersetze die Platzhalter-Icons in `public/` durch echte PNG-Dateien für PWA.
